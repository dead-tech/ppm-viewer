# PPM Viewer
Terminal PPM Viewer written in C++20.

_DISCLAIMER: The viewer is restricted to P3 PPM file format._

## Building with CMake
```shell
$ mkdir build
```
```shell
$ cd build
```
```shell
$ cmake ..
```
```shell
$ make
```

You will find the executable in the build directory you have just created.

## Usage
```shell
$ ppm_viewer <filename.ppm>
```
